# TradeBot 应用功能设计文档
一、应用概述
TradeBot是一个加密货币自动交易机器人应用，通过邀请码注册机制建立代理层级关系，提供交易所连接、自动交易和代理管理功能。
二、用户管理系统
2.1 注册功能
用户必须通过已有用户的邀请码才能注册
注册信息包括：用户名、密码、邮箱、手机号、邀请码
系统自动为新用户生成唯一邀请码
2.2 登录功能
用户名/邮箱 + 密码登录
双重认证选项(可选)
记住登录状态功能
2.3 代理层级系统
上级代理：提供邀请码的用户
直接下级代理：使用用户邀请码注册的用户 
三、主要功能页面
3.1 交易所登录页面
支持自建交易所对接   
显示当前账户基本信息(余额、持仓等)
3.2 交易机器人页面
3.2.1 机器人控制
开启/关闭机器人按钮
运行状态指示器
每日仅允许开启一次
系统参数设置(仅管理员)：
亏损暂停阈值
盈利目标设置
交易频率配置
3.2.2 交易记录
实时同步交易所的交易历史
显示内容：交易对、买入/卖出价格、数量、时间、盈亏
图表化展示交易表现
筛选和搜索功能
3.2.3 机器人运行机制
自动分析市场趋势
自动执行买入/卖出操作
达到亏损阈值时暂停 
后台管理功能：
指定特定币种交易(管理员功能)
控制币价涨跌(管理员功能)
设置盈利目标后自动停止
3.3 代理管理页面
3.3.1 代理概况
总代理人数统计
直接和间接代理分类统计
代理层级关系图表
3.3.2 盈亏分析
所有代理的盈亏总览
按时间段筛选(日/周/月/年)
单个代理详细盈亏情况
导出报表功能
3.3.3 代理等级系统
一级代理：
交易额度上限：200 USDT
基础佣金比例
二级代理：
交易额度上限：500 USDT
提升条件：邀请3-5个有效用户
更高佣金比例
三级代理：
交易额度上限：1000 USDT
提升条件：邀请5-10个有效用户
最高佣金比例
重要规则：无论账户实际资金多少，交易额度仅由代理等级决定

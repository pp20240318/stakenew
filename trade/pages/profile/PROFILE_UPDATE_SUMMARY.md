# 个人中心页面更新总结

## 更新概述
已按照 Lanhu 设计规范重新设计了个人中心页面 (`/pages/profile/profile.vue`)，采用深色主题和现代化 UI 设计。

## 主要变更

### 1. 页面结构重构
采用 Lanhu 的设计风格，包含以下模块：

#### ✅ 状态栏
- 显示时间 (9:41)
- 信号强度图标
- WiFi 图标
- 电池图标

#### ✅ 页面标题
- 居中显示"钱包"标题
- 深色背景，白色文字

#### ✅ 用户资料区域
- 圆形头像 (160rpx × 160rpx)
- 用户名称显示
- UID 显示
- 编辑图标（点击跳转到实名认证）

#### ✅ 余额卡片
- 显示"Your Balance"标签
- 大字号显示余额金额
- 收益率徽章（绿色，带上涨图标）
- 深色卡片背景 rgba(42, 52, 66, 1)

#### ✅ 菜单列表
1. **个人资料** - 跳转到 `/pages/mine/truename`
2. **邀请好友** - 跳转到 `/pages/team/index`
3. **语言** - 跳转到 `/pages/profile/language`，显示当前语言
4. **设置** - 跳转到 `/pages/settings/index`

每个菜单项包含：
- 左侧图标 (48rpx × 48rpx)
- 菜单文字
- 右侧箭头图标

#### ✅ 底部导航栏
- 首页图标 + 文字
- 交易图标 + 文字
- 钱包图标 + 文字（当前激活状态）
- Home Indicator

### 2. 设计风格

#### 配色方案
- **主背景色**: rgba(26, 32, 44, 1) - 深蓝灰色
- **状态栏/导航栏**: rgba(27, 33, 38, 1) - 更深的灰色
- **卡片背景**: rgba(42, 52, 66, 1) - 中等深度灰蓝色
- **主文字**: rgba(237, 242, 247, 1) - 接近白色
- **次要文字**: rgba(113, 128, 150, 1) - 灰色
- **成功色**: rgba(0, 205, 136, 1) - 绿色（用于收益）

#### 字体大小
- 页面标题: 36rpx (粗体)
- 用户名: 36rpx (粗体)
- 余额金额: 48rpx (粗体)
- 菜单文字: 28rpx
- 余额标签: 28rpx
- UID/次要信息: 24rpx
- 导航文字: 20rpx

#### 圆角设计
- 头像: 80rpx (完全圆形)
- 余额卡片: 16rpx
- 菜单项: 12rpx
- 收益徽章: 1000px (胶囊形)

### 4. 功能保留

✅ 保留了所有原有功能：
- `fetchUserData()` - 获取用户数据
- `navigateTo()` - 页面跳转
- `logout()` - 退出登录
- `contactSupport()` - 联系客服
- 用户信息显示
- 余额显示
- 数据加载逻辑

### 5. 响应式设计

- 使用 rpx 单位确保跨设备适配
- 固定底部导航栏
- 支持安全区域 (safe-area-inset-bottom)
- 最小高度 100vh 确保全屏显示

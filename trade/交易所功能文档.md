# COINSP交易所功能文档

## 1. 概述
本文档描述基础现货交易所系统的核心功能模块和业务流程。系统支持用户查看行情、进行简单的买入卖出操作、查看持仓和钱包管理。

## 2. 功能模块

### 2.1 用户模块
- 用户注册/登录
- 身份认证 
- 资金密码设置
- 安全设置（手机验证等）

### 2.2 交易模块
- 现货交易 
  - 市价单
- 交易对管理
  - 交易对列表
  - 交易深度
  - 交易量统计
  - 价格图表

### 2.3 资产管理
- 充币
  - 充值提交
  - 充币记录查询
  - 充币确认
- 提币
  - 提币申请
  - 提币审核
  - 提币记录查询
- 资金流水
  - 交易记录 

### 2.4 持仓管理
- 当前持仓
  - 持仓数量
  - 持仓成本
  - 当前市值
  - 盈亏统计 

### 2.5 行情模块
- 实时行情
- K线图表
- 深度图
- 成交记录 
 

## 3. 业务流程

### 3.1 交易流程
1. 用户登录
2. 选择交易对
3. 查看行情
4. 下单交易（买入/卖出）   

### 3.2 充提币流程
1. 充币
   - 获取充币地址
   - 用户转账
   - 系统确认
   - 入账处理

2. 提币
   - 提币申请 
   - 人工审核 
   - 出账处理
